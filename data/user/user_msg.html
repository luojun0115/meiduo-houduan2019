用户中心------我的消息

当用户点击我的消息时，我们需要用获取用户消息表中（UserMessage）当前用户的消息信息。

在返回数据时，我们需要进行分页返回

请求方式：GET

请求路径：/users/mymessage/

url(r'^mymessage/$', MyMessageView.as_view(), name='mymessage'),

请求参数：无

返回结果：

return render(request, 'usercenter-message.html', {
    'messages': messages,
})

# messages数据分页后的数据对象
