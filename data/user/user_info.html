用户中心------个人资料

1 、获取当前登录用户信息

请求方式：GET

请求路径：/users/info/

url(r'^image/upload/$', UploadImageView.as_view(), name='image_upload'),

携带参数：无

返回结果：

return render(request, 'usercenter-info.html')

# 在模板中可以通过request对象获取用户信息

2、点击修改图片修改头像图片

请求方式：POST

请求路径：/users/image/upload/

请求参数：图片数据

返回结果：

返回json数据

{
  "status":"success",
  "msg":'头像修改成功'
}

3、修改用户信息

点击保存按钮修改用户信息

请求方式：POST

请求路径: /users/info/

url(r'^info/$', UserInfoView.as_view(), name='user_info'),

返回结果：

json数据

# 修改成功
{
  "status":"success"
}

# 修改失败
{
  "status":"error"
}
