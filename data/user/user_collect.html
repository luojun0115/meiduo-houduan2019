项目实训课程讲义

用户中心------我的收藏

需要注意的是，所有数据都是和用户相关的，所以查询条件中需要过滤当前用户信息

课程学校

当用户点击我的收藏，需要先查询用户收藏表（UserFavorite）中课程学校的收藏（根据id查寻）

请求方式：GET

请求路径：/users/myfav/org/ 

url(r'^myfav/org/$', MyFavOrgView.as_view(), name='myfav_org'),

请求参数:无

返回结果：

return render(request, 'usercenter-fav-org.html', {
    'org_list': org_list,
})

# org_list数据形式，
[
  {
    'org_id':收藏学校的id
    'org': 课程学校对像（CourseOrg对象）
  }，
  {
    'org_id':收藏学校的id
    'org': 课程学校对象（CourseOrg类对象）
  }，
]

授课教师

当用户点击授课教师时，需要先查询用户收藏表（UserFavorite）中教师的收藏

请求方式：GET

请求路径：/users/myfav/teacher/

url(r'^myfav/teacher/$', MyFavTeacherView.as_view(), name='myfav_teacher'),

请求参数：无

返回结果：

return render(request, 'usercenter-fav-teacher.html', {
    'teacher_list': teacher_list,
})

# teacher_list数据形式
[
  {
    'teacher_id':收藏教师的id
    'teacher': 收藏教师对像（Teacher类）
  }，
  {
    'teacher_id':收藏教师的id
    'teacher': 收藏教师对像（Teacher类）
  }，
]

公开课程

当用户点击公开课程时，需要先查询用户收藏表（UserFavorite）中公开课的收藏

请求方式：GET

请求路径：/users/myfav/course/

url(r'^myfav/course/$', MyFavCourseView.as_view(), name='myfav_course'),

返回结果：

return render(request, 'usercenter-fav-course.html', {
    'course_list': course_list,
})
# course_list数据形式
[
  {
    'course_id':收藏课程的id
    'course': 收藏课程对像（Course类）
  }，
  {
    'course_id':收藏课程的id
    'course': 收藏课程对像（Course类）
  }，
]
