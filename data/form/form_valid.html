表单验证的使用

当前端把表单数据传递到后端时，我么需要在视图中验证表单数据，我们在视图需要调用表单验证方法来进行验证

from django.shortcuts import render
from django.views import View
from users.forms import RegisterForm
from django.http import HttpResponse


# Create your views here.

class UserRegister(View):
    def get(self, request):
        """
            获取注册页面
        :param request:
        :return:
        """
        # 生成表单对象
        register_form = RegisterForm()
        # 调用模版渲染生成表单
        return render(request, 'register1.html', {'register_form': register_form})

    def post(self, request):
        """
            验证表单数据
        :param request:
        :return:
        """
        # 1、获取前端传递的表单数据
        data = request.POST
        # 2、验证表单数据
        register_form = RegisterForm(data)
        res = register_form.is_valid()  # 验证成功返回True，验证失败返回False

        if res:
            # 验证成功，则执行相应业务逻辑操作，这里就直接返回验证成功后的字段数据
            # 获取验证成功后的字段
            username = register_form.cleaned_data['username']

            return HttpResponse(username)

        # 验证失败，则在注册模板中通过register_form.errors获取错误
        return render(request, 'register1.html', {'register_form': register_form})
  1. register_form.is_valid()方法用来进行表单数据验证
  2. register_form.cleaned_data['username']属性用来获取验证后的字段数据

在模版中获取失败信息进行渲染。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
<form action="/register/" method="post">

    <div>
        {#   register_form.username.label 获取表单中username字段的label选项的名称#}
        <lable> {{ register_form.username.label }}:</lable>
        {#    register_form.username.html_name 获取表单中的username字段名#}
        <input type="text" name="{{ register_form.username.html_name }}">
    </div>
    <div>
        <lable> {{ register_form.password.label }}:</lable>
        <input type="text" name="{{ register_form.password.html_name }}">
    </div>
    <div>
        <lable> {{ register_form.mobile.label }}:</lable>
        <input type="text" name="{{ register_form.mobile.html_name }}">
    </div>
    {% csrf_token %}

    <input type="submit" value="注册">



    {#   如果验证失败，我们可以通不过register_form.errors获取失败信息，然后进行判断 #}
  {#   在视图get方法中，我们返回的register_form对象中是没有错误信息的，则if判断失效，不会显示错误内容 #}
    <div>
        {% if register_form.errors.username  or register_form.errors.mobile%}
            用户名或手机号错误
        {% endif %}

    </div>

</form>
</body>
</html>
