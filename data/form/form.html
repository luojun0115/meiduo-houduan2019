表单模板渲染的使用

我们在需要注册视图调用模版来渲染生成注册表单页面

from django.shortcuts import render
from django.views import View
from users.forms import RegisterForm
# Create your views here.
class UserRegister(View):

    def get(self,request):
        """
            获取注册页面
        :param request:
        :return:
        """
        # 生成表单对象
        register_form=RegisterForm()
        # 调用模版渲染生成表单
        return render(request,'register.html',{'register_form':register_form})

模版的渲染，在模板中创建register.html来渲染数据

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
<form action="/register/" method="post">
  {#获取视图中传递过来的register_form对象直接进行加载渲染#}
    {{ register_form }}
    {% csrf_token %}
    <input type="submit" value="注册">

</form>
</body>
</html>

配置子路由和全局路由

# 子路由配置
from django.conf.urls import url
from . import views
urlpatterns = [
    url(r'^register/', views.UserRegister.as_view()),
]


#  全局路由配置
from django.conf.urls import url,include
from django.contrib import admin

urlpatterns = [
    url(r'^admin/', admin.site.urls),
    url(r'^', include('users.urls')),
]

我们在实际中需要对表单样式进行控制，这是我们就需要对模板的展示样式进行控制，我们就不能直接使用来生成表单，我们需要按照如下方式生成表单:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
<form action="/register/" method="post">

    <div>
        {#   register_form.username.label 获取表单中username字段的label选项所指定的名称  #}
        <lable> {{ register_form.username.label }}:</lable>
        {#    register_form.username.html_name 获取表单中的username字段名  #}
        <input type="text" name="{{ register_form.username.html_name }}">
    </div>
    <div>
        <lable> {{ register_form.password.label }}:</lable>
        <input type="text" name="{{ register_form.password.html_name }}">
    </div>
    <div>
        <lable> {{ register_form.mobile.label }}:</lable>
        <input type="text" name="{{ register_form.mobile.html_name }}">
    </div>
    {% csrf_token %}

    <input type="submit" value="注册">

</form>
</body>
</html>
