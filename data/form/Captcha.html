验证码模块

在我们进行登录或注册时需要在表单中生成一个验证码，这是我们使用Django Simple Captcha模块配合Django的表单系统自动完成验证码图片的生成和验证。

官方文档地址：

https://django-simple-captcha.readthedocs.io/en/latest/usage.html#

展示效果:

Django Simple Captcha的安装与使用

1、下载安装

pip install  django-simple-captcha

2、添加注册到Django应用

INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'users.apps.UsersConfig',

    'captcha' # 图片验证码模块
]

3、配置全局路由

urlpatterns = [
    url(r'^admin/', admin.site.urls),
      # user应用路由
    url(r'^', include('users.urls')),
    # 图片验证码路由
    url(r'^captcha/', include('captcha.urls')),
]

4、迁移数据库生成验证码表

python manage.py migrate

5、在注册表单类中增加验证码字段

from captcha.fields import CaptchaField
from django import forms


class RegisterForm(forms.Form):
    """
        用户注册表单
    """
    username = forms.CharField(max_length=20, min_length=5, required=True,label='用户名')
    password = forms.CharField(max_length=20, min_length=8, required=True,label='密码')
    mobile = forms.CharField(max_length=11, min_length=11,required=True,label='手机号')
    # 增加验证码字段
    captcha = CaptchaField(label='验证码')

6、在模版中渲染生成验证码图片

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
<form action="/register/" method="post">

    <div>
        {#   register_form.username.label 获取表单中username字段的label选项的名称#}
        <lable> {{ register_form.username.label }}:</lable>
        {#    register_form.username.html_name 获取表单中的username字段名#}
        <input type="text" name="{{ register_form.username.html_name }}">
    </div>
    <div>
        <lable> {{ register_form.password.label }}:</lable>
        <input type="text" name="{{ register_form.password.html_name }}">
    </div>
    <div>
        <lable> {{ register_form.mobile.label }}:</lable>
        <input type="text" name="{{ register_form.mobile.html_name }}">
    </div>


    {#   通过register_form.captcha 渲染生成图片验证码#}
     <div>
        <lable> {{ register_form.captcha.label }}:</lable>
         {{ register_form.captcha }}
    </div>

    {% csrf_token %}

    <input type="submit" value="注册">



    {#    如果验证失败，我们可以通不过register_form.errors获取失败信息，然后进行判断#}
    <div>
        {% if register_form.errors.username  or register_form.errors.mobile%}
            用户名或手机号错误
        {% endif %}

    </div>

</form>
</body>
</html>
