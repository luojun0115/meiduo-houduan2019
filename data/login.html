用户登录

1、获取登录页面

请求方式：GET

请求路径：login/

# 路由名为login
url(r'^login/$', LoginView.as_view(), name='login'),

请求参数：无

返回结果：

return render(request, 'login.html')

2、完成登录请求

请求方式：POST

请求路径：login/

携带参数：

参数 类型 说明
email str 邮箱
password str 密码

返回结果：

登录成功

return HttpResponsePermanentRedirect(reverse('index'))

登录失败：

return render(request, 'login.html', {'form_errors': login_form.errors})
