忘记密码

当用户在登录页面点击忘记密码时获取忘记密码页面

1 、获取忘记密码页面

请求方式：GET

请求路径：/forget/

# 忘记密码
 url(r'^forget/$', ForgetPwdView.as_view(), name='forget_pwd'),

请求参数：无

返回结果：

  return render(request, 'forgetpwd.html', {'forget_form': forget_form})

2、发送邮件

当用户点击提交时，验证表单数据，然后发送邮件到用户邮箱

请求方式：POST

请求路径：/forget/

携带参数：

参数 类型 说明
email str 邮箱
Captcha str 验证码

返回结果：

验证数据成功返回

return render(request, 'send_success.html')

验证数据失败返回：

 return render(request, 'forgetpwd.html', {'forget_form': forget_form})

在这里需要用到发送邮件的操作，在链接中需要包含用户的加密信息，方便下一步修改密码时进行验证
